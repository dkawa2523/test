バッチ型（浸漬）洗浄で「ウェハー位置ごとのエッチングアマウント（膜減り）」を理論計算する場合、**反応（界面）＋液膜（境界層）拡散＋バス（槽）全体の時空間変動**が同時に効きやすいのが “バッチ特有” の難しさです。特に**多数枚を同一バスで並列処理**するため、流れ・温度・濃度のわずかな不均一が、そのまま位置依存のエッチ量差になります（実際にバッチ槽の流れ不均一がエッチレート差につながることが指摘されています）。 ([COMSOL][1])

以下に、**反応・液膜拡散・バッチ運用**で「モデルに入れるべき物理（考慮事項）」を、**どのシーンで必要か**と**必要度（1–5）**つきで整理しました（基本は**解析解／ODE（0D〜1D）**で扱える形を意識）。

## 最低限の “ODE/解析解ベース” 推奨骨格（バッチ洗浄のエッチ量をまず出す）

あなたが「別シミュレーションからウェハ面上の濃度・フラックス分布」を入力できる前提でも、**最初に破綻しにくい**骨格は次です。

1. **局所（ウェハ面）**

* 入力がフラックス (j(x,y,t)) なら：
  [
  \Delta h(x,y)=\frac{M}{\rho,\nu}\int_0^{t_{\text{end}}} j(x,y,t),dt
  ]
* 入力が濃度（境界層外） (C_b(x,y,t)) なら：
  [
  j = k_m(x,y,t),\bigl(C_b - C_s\bigr),\quad j = r_s(C_s,T,\mathrm{pH})
  ]
  を連立して (C_s) を消し、(j) を得る（直列抵抗の考え方）。 ([idc-online.com][2])

2. **バス全体（0Dまたはゾーン0D）**
   [
   \frac{dC_b}{dt}= -\frac{A_{\text{tot}}}{V}\nu,\overline{j}(t);-;k_{\text{decay}}C_b;+;\text{補給/入替}
   ]
   O₃等は分解項が効く（pH・温度依存）。 ([Absolute Ozone][6])
   生成物は蓄積・再析出（浴寿命）を別状態変数で持つと“バッチらしさ”が出ます。 ([entegris.com][3])

3. **位置依存（バッチ特有）**

* まずは **(k_m(x,y))** を「別シミュレーションの流速場→相関式で変換」または「学習で同定」
* さらに必要なら、**カセットのウェハ番号方向（z）**をゾーン化して (C_{b,z}(t))、(k_{m,z}) を持つ（多枚数バッチの最小表現）

## 最低限の “ODE/解析解ベース” 推奨骨格（バッチ洗浄のエッチ量をまず出す）

あなたが「別シミュレーションからウェハ面上の濃度・フラックス分布」を入力できる前提でも、**最初に破綻しにくい**骨格は次です。

1. **局所（ウェハ面）**

* 入力がフラックス (j(x,y,t)) なら：
  [
  \Delta h(x,y)=\frac{M}{\rho,\nu}\int_0^{t_{\text{end}}} j(x,y,t),dt
  ]
* 入力が濃度（境界層外） (C_b(x,y,t)) なら：
  [
  j = k_m(x,y,t),\bigl(C_b - C_s\bigr),\quad j = r_s(C_s,T,\mathrm{pH})
  ]
  を連立して (C_s) を消し、(j) を得る（直列抵抗の考え方）。 ([idc-online.com][2])

2. **バス全体（0Dまたはゾーン0D）**
   [
   \frac{dC_b}{dt}= -\frac{A_{\text{tot}}}{V}\nu,\overline{j}(t);-;k_{\text{decay}}C_b;+;\text{補給/入替}
   ]
   O₃等は分解項が効く（pH・温度依存）。 ([Absolute Ozone][6])
   生成物は蓄積・再析出（浴寿命）を別状態変数で持つと“バッチらしさ”が出ます。 ([entegris.com][3])

3. **位置依存（バッチ特有）**

* まずは **(k_m(x,y))** を「別シミュレーションの流速場→相関式で変換」または「学習で同定」
* さらに必要なら、**カセットのウェハ番号方向（z）**をゾーン化して (C_{b,z}(t))、(k_{m,z}) を持つ（多枚数バッチの最小表現）

## 使い分けの目安（どこまで入れるべきか）

* **最初に入れるべき（=必要度5〜4）**：
  **(2)(3)(4)(5)(7)(8)(9)(12)(18)**
  → バッチの“ムラ”と“時間変化”の大半がここに載ります。特に槽内流れの非一様がレート差につながる点は重要。 ([COMSOL][1])
* **音響併用なら優先度が上がる（=必要度3→4）**：
  **(10)(17)(19)**（音響流・キャビテーション・複雑流れ） ([PCT Systems][4])
* **欠陥/局所ムラ（斑点）を潰したい時**：
  **(9)(12)(16)**（気泡マスク、生成物再析出、電気化学）
